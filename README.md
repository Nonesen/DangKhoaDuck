# DangKhoaDuck
Đây là nơi mình lưu trữ và chia sẻ mã nguồn cho các dự án học tập hoặc thực hành, cách quản lý phiên bản, và cộng tác với người khác. Nếu bạn cũng đang học lập trình, hãy cùng khám phá và đóng góp nhé!

This is where I store and share source code for learning or practice projects, version control, and collaborate with others. If you are also learning to code, please explore and contribute!
